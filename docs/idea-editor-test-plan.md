# Idea Editor 测试计划

> 目标：验证 Source 模式（Markdown 编辑）和 Preview 模式（富文本编辑）的功能完整性，
> 以及两种模式之间切换时内容的一致性和稳定性。

---

## 一、术语约定

| 术语 | 含义 |
|------|------|
| Source 模式 | textarea 直接编辑 markdown 源码 |
| Preview 模式 | contentEditable 富文本编辑，所见即所得 |
| 切换 | Cmd/Ctrl + / 或点击工具栏按钮 |
| 内容一致 | 切换前后 markdown source 字节级相同，渲染结果视觉一致 |
| 光标保持 | 切换前后光标停留在同一逻辑位置（同一段落、同一偏移） |
